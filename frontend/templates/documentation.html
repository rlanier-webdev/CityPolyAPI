Documentation

This API provides endpoints to retrieve data about the historic City-Poly rivalry game. All data endpoints require an API key passed via the X-API-Key header.

1Register — Create an account with your email and password.
2Login — Receive a bearer token.
3Create API Key — Use the bearer token to generate an API key.
4Use the API Key — Pass it as X-API-Key on all data requests.
